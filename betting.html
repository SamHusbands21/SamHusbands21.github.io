How it works

A quick overview of the model and how value bets are identified.

Elo ratings

Each team carries a standard Elo rating updated after every result (W=1, D=0.5, L=0, K=20). A 75-point home advantage offset is applied to the effective home rating when computing expected scores.

xG-Elo

A second Elo system is updated using expected goals (xG) share as the "score" — xGhome / (xGhome + xGaway). This is far less noisy than actual results and better reflects underlying performance.

Features

Match features include both Elo deltas, rolling points-per-game (last 5 and 10 matches), rolling xG for/against, actual goals for/against, days of rest, and head-to-head win rate from the last five meetings.

Model

An XGBoost classifier and a calibrated Random Forest are trained separately on seasons 2014/15–2021/22 using walk-forward time-series cross-validation. Final probabilities are an average of both calibrated models.

Value bets

A bet is flagged when model probability × Betfair decimal odds > 1.05, indicating at least a 5% expected edge. Kelly fraction is also shown as a suggested stake size (capped at 25% of bankroll).

Test set performance

The model is evaluated strictly on held-out seasons 2022/23–2024/25 (never seen during training). Metrics include log loss, Brier score, calibration curves, and flat/Kelly staking ROI vs. a naive always-favourite baseline.
